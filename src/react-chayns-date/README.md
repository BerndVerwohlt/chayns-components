# Date-Component #

The **Date** - Component is part of the **chayns-component** package. You can install the whole package with the following statement:

    npm install -S chayns-components@latest


## Usage ##

First of all import the component to a component of your own project.

```jsx harmony
import { Date } from 'chayns-components';
```

You can use the date-component like in the [example](https://github.com/TobitSoftware/chayns-components/blob/master/examples/react-chayns-date/Example.jsx).
